ชุดโครงการชุมชนที่สร้างขึ้นบน Cipher

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Cipher ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-Cipher](https://github.com/awesome-opencode/awesome-Cipher) และ [Cipher.cafe](https://opencode.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
